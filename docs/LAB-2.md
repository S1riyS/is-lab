# Лабораторная работа №2

## Формат файла импорта

- Тип: `json`
- Формат: 
    ```json
    {
        "entityType": "ENTITY_TYPE",
        "data": [
            {
                "score": 100,
                "grade": "5A",
                "details": {...}
            },
            ...
        ]
    }
    ```

## Ограничения, проверяемые на программном уровне 

1. Допустимый размер скидки в зависимости от типа билета (VIP, BUDGETARY, ...)
2. Coordinates (x, y) принадлежит заданной области (как в лабах по Web'у)
